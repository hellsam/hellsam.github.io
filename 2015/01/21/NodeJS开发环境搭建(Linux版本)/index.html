开发前准备

既然是Linux版那咱怎么的也得整个Linux系统吧，你可以装个虚拟机可以用vmware或者是免费开源的VirtualBox
壕不差钱的话可以直接上阿里云买个服务器(国内的还可以选择青云，腾讯云，国外的话推荐linode, digitalocean)

本文测试系统是CentOS 6.5 64位 和 Ubuntu 12.04 64位

开始搭建

git 安装

Ubuntu

CentOS

好了，已经安装成功了

安装 n

n 是NodeJS的多版本管理工具

n 已经安装成功了，但是当前还没有安装任何nodejs版本

安装NodeJS

当前node最新版本为0.11.15

安装node 0.11.15版本

Ubuntu 可能需要先安装一下curl

安装完成后， 输入 n 会显示当前已安装的所有node版本，用键盘方向盘上下键可以切换不同的node版本

输入node就可以进入node命令行交互模式

按两次Ctrl+C 可以退出node命令行模式
